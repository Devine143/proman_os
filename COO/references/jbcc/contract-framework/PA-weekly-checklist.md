# Principal Agent Weekly Checklist

Use this checklist to ensure all JBCC duties are being fulfilled systematically.

---

## Weekly Review (Every Monday)

### Contract Administration
- [ ] Review all pending PA Instructions - any requiring follow-up?
- [ ] Check if any new instructions need to be issued
- [ ] Review updated construction program from Contractor
- [ ] Verify all instructions from last week were issued in writing

### Quality & Inspections
- [ ] Schedule this week's site inspection visits
- [ ] Review any outstanding rejections - has Contractor rectified?
- [ ] Check if any materials require approval this week
- [ ] Review snag list progress (if post-PC)

### Time Management
- [ ] Review any new EOT claims received
- [ ] Check if any EOT determinations are due (10-day deadline)
- [ ] Assess current program vs actual progress - slippage?
- [ ] Update master schedule with any granted EOTs

### Cost Management
- [ ] Review any new variation requests
- [ ] Price any pending variations
- [ ] Update cumulative variations register
- [ ] Check budget status vs forecast

### Payment Certification
- [ ] Check if Contractor payment claim due this week
- [ ] Schedule site verification for payment claim
- [ ] Review QS valuation (if applicable)
- [ ] Ensure payment certificate issued within 10 working days

### Communication
- [ ] Prepare site meeting agenda
- [ ] Review action items from last meeting - any overdue?
- [ ] Draft weekly status report for Employer
- [ ] Follow up on pending RFIs or submissions

### Risk & Issues
- [ ] Update risk register with new/changed risks
- [ ] Escalate any critical issues to Employer
- [ ] Review Contractor performance - any concerns?
- [ ] Check if any disputes brewing - early intervention needed?

---

## Monthly Review (First Monday of Month)

- [ ] Review project KPIs (SPI, CPI, quality metrics)
- [ ] Update Employer on overall project health
- [ ] Review consultant performance and coordination
- [ ] Check Professional Indemnity insurance is current
- [ ] Archive previous month's correspondence and records
- [ ] Review contract milestones - any approaching?

---

## Project Milestone Checks

### At Practical Completion
- [ ] Conduct thorough PC inspection
- [ ] Prepare comprehensive snag list
- [ ] Issue Practical Completion Certificate (if ready)
- [ ] Certify retention release (per contract)
- [ ] Start defects liability period tracking

### At Final Completion
- [ ] Conduct final inspection
- [ ] Verify all snags complete
- [ ] Issue Final Completion Certificate
- [ ] Certify final retention release
- [ ] Prepare final account with QS
- [ ] Issue Final Payment Certificate
- [ ] Obtain as-built drawings and O&M manuals
- [ ] Document lessons learned

---

## Red Flag Monitoring

**Watch for these issues weekly:**
- [ ] Contractor falling behind program (> 5 days)
- [ ] Quality issues recurring
- [ ] Delayed responses from consultants
- [ ] Budget approaching variation threshold
- [ ] Safety incidents or near misses
- [ ] Contractor cash flow concerns
- [ ] Employer changes affecting scope
- [ ] Unresolved disputes escalating

---

**Print and use weekly. Update as needed based on project-specific requirements.**
